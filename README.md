# Avant-Garde Enterprise Website

This is a [Next.js](https://nextjs.org) project for Avant-Garde Enterprise.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Modern, sleek design with cyberpunk aesthetic
- Supabase authentication
- Client portal with secure dashboard
- Blog system
- Contact forms

## Deployment

The site is deployed on Vercel.
